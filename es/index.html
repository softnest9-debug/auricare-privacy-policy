1. Información General

Esta Política de Privacidad describe los principios de procesamiento de datos en la aplicación móvil AuriCare – Hearing Aid App (en adelante, la "Aplicación"). Nos preocupamos por la privacidad de los usuarios y limitamos el procesamiento de datos exclusivamente a lo necesario para el funcionamiento de la Aplicación.

La Aplicación funciona localmente en el dispositivo: el sonido se procesa directamente en la memoria del teléfono y no se transmite ni se almacena en ningún servidor externo.

2. Alcance y Método de Procesamiento de Datos

  • La Aplicación no recopila, no perfila y no almacena de forma permanente los datos personales de los usuarios.
  • El sonido capturado se utiliza exclusivamente para la amplificación/limpieza y se procesa localmente.
  • No grabamos archivos de audio y no los transmitimos fuera del dispositivo del usuario.
  • La Aplicación no accede a la ubicación del usuario ni a los datos de otras aplicaciones.

3. Permisos Requeridos por la Aplicación

Para su correcto funcionamiento, la Aplicación solicita los siguientes permisos del sistema Android:

  • Grabar audio (micrófono) – exclusivamente para capturar el sonido del entorno y procesarlo en tiempo real en el dispositivo.
  • Buscar/conectar con dispositivos cercanos (Bluetooth) – para el manejo de auriculares y accesorios de audio.
  • Notificaciones – para mostrar una notificación persistente que informa que la Aplicación está activa y funcionando en segundo plano.
  • Desactivar la optimización de la batería (opcional, pero recomendado) – La Aplicación puede solicitar que se desactive la optimización de la batería para ella en el sistema Android. Esto es necesario para garantizar un funcionamiento estable e ininterrumpido del servicio de procesamiento de sonido cuando la pantalla del dispositivo está apagada.

La denegación de los permisos clave (micrófono, Bluetooth, notificaciones) impedirá el uso de las funciones básicas de la Aplicación.

4. Período de Prueba y Pagos

La Aplicación ofrece un período de prueba gratuito de 7 días. Una vez finalizado, el usuario puede elegir una suscripción o una compra única para seguir utilizando la funcionalidad completa de la Aplicación.

Los pagos son gestionados por Google Play. En este ámbito, se aplican las políticas de privacidad y los términos de servicio de Google. Detalles: Política de Privacidad de Google.

5. Anuncios y Recopilación de Datos

La versión gratuita de la Aplicación (licencia Basic) muestra anuncios proporcionados por Google a través del servicio AdMob. En este contexto, la Aplicación utiliza las siguientes tecnologías:

  • SDK de Google Mobile Ads: Esta biblioteca puede recopilar y procesar datos como el identificador de publicidad único del dispositivo (ID de publicidad), la dirección IP, información sobre las interacciones con los anuncios y datos de ubicación aproximada para ofrecer contenido publicitario personalizado y relevante.
  • SDK de la Plataforma de Mensajería de Usuario (UMP): En las regiones sujetas al RGPD (EEE y Reino Unido), la Aplicación utiliza esta biblioteca para obtener el consentimiento de los usuarios para la recopilación y el uso de datos con fines de personalización de anuncios. El usuario tiene la posibilidad de gestionar sus preferencias a través del formulario de consentimiento.

No tenemos acceso directo a los datos recopilados por Google. El uso de estos datos está sujeto a la política de privacidad de Google. Para obtener más información sobre cómo Google utiliza los datos, consulte el documento: Privacidad y Condiciones de Google – Publicidad.

8. Seguridad

Dado que la Aplicación no almacena datos personales ni los transmite a servidores externos, el riesgo de una violación de la privacidad se minimiza. Todos los datos de audio se procesan localmente en el dispositivo del usuario.
